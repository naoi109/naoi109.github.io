辻村小説公園

UX / UI / WEB

小説家である辻村七子先生のWEBサイトを制作させていただきました。ブログや著書、執筆されているシリーズ小説のサイドストーリー等を閲覧できます。

「文章」がメインとなるサイトなので、WEBサイトによくあるアニメーション等はすべて削ぎ、文章が読みやすくなる構成となるよう意識しています。また、「公園」というタイトル、また辻村先生のお人柄からあたたかなイメージのデザインにまとめています。

小説に出てくる異国の情景がとても美しいこと、また辻村先生ご自身がよく海外を旅していらっしゃることから、キービジュアル画像には好きなお写真を自由に設定できる仕掛けも制作いたしました。
